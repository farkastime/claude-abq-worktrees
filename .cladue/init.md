This repository is designed to house resources for a micro workshop to train students to use git worktrees for parallel development using Claude Code. I want the repo to house a website with instructions and architecture to launch locally so that users can create multiple worktrees and make edits on different branches. I want to show a few different things during this workshop:

1. The basics of git worktrees. Demo git add worktree and show a workflow of making updates on main, updates on worktree1, then merging worktree1 into main.
2. Spinning up worktrees with claude --worktree.
3. Adding hooks for AddWorktree and RemoveWorktree.
4. Spinning up multiple webservers on different worktrees.
